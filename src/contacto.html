Frequently Asked Questions

How does Files Organizer Work?

Files Organizer automatically scans your Downloads folder and organizes files based on their types into appropiate folders. You can customize the organization rules in the Config section.

Is Files Organizer free to use?

Yes! Files Organizer is completely free and open-source. You can contribute to the project on our GitHub repository.

Can I customize the organization rules?

Absolutely! Visit the Config page to customize file organization rules, create custom categories, and set up automated organization schedules.
